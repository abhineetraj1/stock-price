# Stock chart
This platform shows you daily stock prices of 1900+ NYSE listed companies in line graph. I used alphavintage API to get details.

## Features:-

*	User login and sign up
*	Alphavintage API integration
*	Daily price tracker of every stock
*	1900+ NYSE companies
